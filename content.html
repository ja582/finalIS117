what i learned in IS-117

Git & GitHub

This year in IS-117 I learned the how to commit, push, pull, and merge in Git with my partner. This taught me how to collaborate properly and manage code efficiently. By learning how to use Git I can now upload my code somewhere safe remotely and share it with others who want to collaborate with me.

I've never used Git before this class. I've been on GitHub before. But actually interacting with and learning Git taught me a whole new way of development. It is extremely handy for wanting to quickly contribute code. Pushing, pulling, commits, and merging is very intuitive.

Bootstrap

In IS-117 we learned the famous Bootstrap framework. I've never used it until now, but always heard of it. Learning it was such a breeze. While my website isn't super pretty or flashy, getting the basic gist of a layout in Bootstrap is extremely nice. Picking out parts for my website layout is way more fun than painstakingly try to get the correct float.

Before this class I already knew basics in HTML/CSS from teaching it myself in 7th grade. It was always a pain in the ass for me personally because I hated dealing with floats, margins, etc. Using Bootstrap saved a lot of time for developing this website. Modern web development has come a long way since 2007 for me.

Quick Facts about Bootstrap

  • Started in 2011.
  • It is completely open-source.
  • Created by Twitter Employees, but completely separate now.
  • Third-most-starred project on GitHub.
  • Bootstrap 4 was announced in 2014, finally released in 2018.
