My name is Joe and I took IS-117 over the Spring '19 semester at NJIT. It's been enjoyable and hope to apply what I learned later down the road in my college career. This is my portfolio website final project for IS-117.

Why take IS-117?

I took this class as a requirement for my Web Development specialization for my IT degree. IS-117 is a class about the basics of web development and design. Most things in the class I knew, except for Git and Bootstrap.

In this class we went over:

  • Git and GitHub
  • History of the Internet
  • Linux Commands & vi
  • How to configure an IDE to upload to a remote
  • Bootstrap, CSS, and HTML
